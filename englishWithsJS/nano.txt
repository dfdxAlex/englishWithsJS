Открытие файла:
nano filename

Сохранение файла:
Ctrl + O  — сохранить файл (подтвердите имя файла, нажмите Enter)

Выход из редактора:
Ctrl + X  — выйти из редактора

Вырезание, копирование и вставка:
Ctrl + K  — вырезать текущую строку (или выделенный текст)
Ctrl + U  — вставить вырезанный или скопированный текст
Alt + 6   — скопировать выделенный текст

Выделение текста:
Ctrl + ^  — начать выделение текста

Поиск и замена:
Ctrl + W  — найти текст
Ctrl + \  — найти и заменить текст

Отмена и повтор действий:
Ctrl + Z  — свернуть nano
Ctrl + _  — перейти на строку и столбец (после команды введите номера)

Перемещение по тексту:
Ctrl + A  — в начало строки
Ctrl + E  — в конец строки
Ctrl + Y  — прокрутить вверх на одну страницу
Ctrl + V  — прокрутить вниз на одну страницу

Удаление текста:
Ctrl + K  — удалить (вырезать) текущую строку
Backspace — удалить символ перед курсором
Ctrl + D  — удалить символ под курсором

Справка:
Ctrl + G  — открыть справку

Отмена/повтор с Meta:
Alt + U  — отменить действие
Alt + E  — повторить действие
